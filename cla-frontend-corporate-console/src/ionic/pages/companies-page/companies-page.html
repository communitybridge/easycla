Welcome to the Corporate console for EasyCLA

For setting up corporate CLA Manager for a new company or add a new CLA Manager to an existing company, click the button below to get started.

Once you select the company and project for which you want to be a CLA Manager your request will be sent to your company’s initial CLA Manager for approval.

If you don’t see your company in “Company Name” field please type it. You can type it and you will be added as a CLA Manager for that company

If you do not see your project in "Project Name" filed please type it. The project will be added to the database.
